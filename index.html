Hi, I'm Stephanie and I'm a Front-End Web Developer based in London, UK.

A little bit about me

Over 10 years ago I started up my first ever blog which was all about Manchester United and my love of football. I started off using Blogger and a free web domain but I learnt very quickly that I wanted my own website that I owned. After buying my domain and setting it up on Wordpress, I was introduced into the massive world of templates and widgets.

I very quickly became frustrated at using free templates, I wanted to edit them but it never worked in the way I wanted. So, I decided to try it myself. Initially it was very scary, all I saw was a load of code and I had no idea what it meant but I persevered and eventually I ended up creating my own website that I was happy with.

I enjoyed creating my own website so much I decided to learn code properly and that's where my journey as a Web Developer started - from someone who knew nothing about code to someone who is a self-taught developer. My journey hasn't been easy but it's been rewarding and now that's where I can help you. My aim is to help people have their own websites which they're proud of and exactly how they want them to be.
